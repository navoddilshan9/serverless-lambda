This backend server mainly focus to use AWS services. I used, 

01) AWS route 53
02) AWS s3 bucket
03) AWS cognito
04) AWS lambda functions
05) AWS API gateway
06) AWS loadbalancer and auto scaling groups
07) Mongodb

This is the high level architechture the entire application.
